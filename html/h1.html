La France, où immigration rime avec désinformation

Mode Quizz

Bienvenue dans le mode Quizz !

Cette partie histoire vous est présentée sous forme de quiz composé de 5 questions clés. Elle vous permettra de vous tester ludiquement sur l'immigration dans ses grands axes. Chaque question est composée d'une vidéo et d'une question. Une seule réponse est correcte et, pour chaque question il est possible une fois que vous y avez répondu d'en savoir plus tout en revenant au quizz, mais quoiqu'il en soit toutes les ressources sont disponibles dans le mode Sources !
